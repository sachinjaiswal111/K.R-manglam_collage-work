# OS Lab Assignment 2 - System Simulation

## Description
Simulation of system startup, process creation, and termination using Python multiprocessing.

## Files
-  - Main Python script
-  - Generated log file
-  - Assignment report

## How to Run
```bash
cd code
python3 system_simulation.py
```

## Features
- System startup sequence simulation
- Multiple concurrent processes
- Detailed logging with timestamps
- System shutdown sequence
